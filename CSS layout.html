Etiqueta SPAN

permite agrupar varios elementos en línea seguidos dentro de un mismo bloque

En este ejemplo:
djfhfdbghjfdgbvhdvgfvgf
dfjnhdjgfudgfyudgfdsjgfsdgfsdjh
kjsfbsdvfsdjkfsgfvgfkgfvfsdjkfsd
subrayaremos todo este bloque de texto

Box model

Básicamente, es un módulo CSS que define cajas rectangulares.

Dentro del boxmodel nos encontramos: content, padding, border, margin


Padding

Marca el margen para los 4 lados, es más práctico para evitar ir lado por lado con las otras propiedades


border

La usamos para añadir valores del ancho, el estilo y color de borde de los elementos, solo puede establecer 4 bordes y solo un ancho y estilo de borde.

Margin

Marca el espacio de relleno de los lados de un elemento. Su área es el esapcio entre el contenido y su borde
