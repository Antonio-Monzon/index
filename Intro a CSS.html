SELECCIÓN POR ETIQUETA// (A.M)


Selecciona todos los elementos de la página cuya etiqueta HTML coincide con el valor del selector.

SELECCIÓN POR ID atrbuto"id"// (A.M)

En un documento HTML, los selectores de ID de CSS buscan un elemento basado en el contenido del atributo id.
El atributo ID del elemento seleccionado debe coincidir exactamente con el valor dado en el selector.

SELECCIÓN POR CLASES, atributo"class"// (A.M)

En un documento HTML, los selectores de clase buscan un elemento
basado en el contenido de su atributo class.
